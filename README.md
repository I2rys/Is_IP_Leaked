## Installation
Github:

    git clone https://github.com/I2rys/Is_IP_Leaked

NPM Packages:

    npm i puppeteer && npm i chalk && npm i moment && npm i is-ip
    
## Usage

    node index.js <ip>

 - ip - The IP to search in multiple search engines.
